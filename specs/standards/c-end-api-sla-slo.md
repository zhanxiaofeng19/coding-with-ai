# C 端接口 SLA/SLO 定义

## SLO 目标

### 可用性
| 等级 | 目标 | 月度允许停机 |
|------|------|------------|
| 核心接口 | 99.95% | 21.9 分钟 |
| 重要接口 | 99.9% | 43.8 分钟 |
| 一般接口 | 99.5% | 3.65 小时 |

### 延迟
| 等级 | P50 | P99 | P999 |
|------|-----|-----|------|
| 核心接口 | ≤ 50ms | ≤ 200ms | ≤ 500ms |
| 重要接口 | ≤ 100ms | ≤ 500ms | ≤ 1s |
| 一般接口 | ≤ 200ms | ≤ 1s | ≤ 3s |

### 错误率
| 等级 | 阈值 |
|------|------|
| 核心接口 | < 0.01% |
| 重要接口 | < 0.1% |
| 一般接口 | < 1% |

## 接口分级

### 核心接口（P0）
- 用户登录/鉴权
- 会员信息查询
- 下单
- 支付

### 重要接口（P1）
- 商品/菜单查询
- 门店查询
- 优惠券领取/核销
- 积分查询

### 一般接口（P2）
- 会员等级详情
- 权益列表
- 活动页面
- 历史记录

## 容量基线

| 指标 | 日常 | 峰值 |
|------|------|------|
| QPS | 2,000 | 10,000 |
| 峰值倍数 | 1x | 3~5x |
| 存储增速 | 10GB/月 | - |

## 降级触发条件

| 条件 | 动作 |
|------|------|
| 错误率 > 5% | 自动降级非核心功能 |
| P99 > 2s | 触发限流 |
| 依赖超时率 > 10% | 触发熔断 |
| CPU > 80% | 自动扩容 |
