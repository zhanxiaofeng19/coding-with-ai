# 安全规范

## 鉴权与授权
- 所有接口必须鉴权
- Token 方案（JWT / OAuth2）
- RBAC 权限模型
- 数据权限：用户只能访问自己的数据
- 最小权限原则

## 输入校验
- 所有外部输入必须校验
- 参数化 SQL，禁止拼接
- 防 XSS：输出编码
- 防 CSRF：Token 校验
- 文件上传：校验类型和大小

## 敏感数据
- 手机号：138****8888
- 身份证：110***********1234
- 银行卡：6222 **** **** 1234
- 密码：BCrypt 加密存储
- 日志自动脱敏
- HTTPS 传输

## 密钥管理
- 禁止代码中硬编码
- 配置中心加密存储
- 定期轮换
- 最小权限分配

## 审计
- 登录/退出
- 数据修改
- 权限变更
- 配置变更
- 独立存储，不可篡改

## 接口安全
- 限流防刷
- 防重放（时间戳 + 签名）
- 敏感操作二次验证
- 不暴露内部实现
